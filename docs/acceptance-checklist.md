# 当前工具全面验收清单

本文档用于验收当前 `v0.2.82` 版本的 AI 画布工具。验收目标不是证明每个历史功能都存在，而是确认用户能沿当前主路径完成一次短剧生产闭环，并能定位阻断风险。

上线前总判断见 `docs/release-acceptance-report.md`。本文档保留逐项验收清单和取证记录。

## 验收结论口径

- 通过：P0 / P1 项全部通过，P2 项无明显主流程阻断，文档和版本记录能解释当前状态。
- 有条件通过：P0 项通过，存在 P1 风险但有明确绕行路径，且不影响继续人工试用。
- 不通过：任一 P0 项失败，或 P1 项导致项目、分集、画布、素材、AI 配置、生图 / 视频承接任一主路径无法继续。

## 验收前置

1. 确认当前分支干净：`git status --short` 无输出。
2. 确认版本一致：`VERSION`、`web/package.json`、`web/desktop/app/package.json` 均为 `0.2.82` / `v0.2.82`。
3. 启动前端：在 `web/` 下运行 `npm run dev`，默认访问 `http://localhost:3000`。
4. 如需验证后端代理和管理后台，启动 Go 后端并确认 `/api/*` 代理可用。
5. 准备素材：至少 3 张图片、1 个视频、1 个音频、1 段短剧剧本文本、1 份 AI 配置包 JSON。

## P0 主流程验收

### 1. 核心入口与导航

- 入口：`/`、`/projects`、`/image`、`/prompts`、`/assets`、`/canvas`、`/canvas/:id`。
- 操作：
  1. 打开 `/`，未登录时确认进入 `/login?redirect=/projects`；登录后应回到项目工作台。
  2. 打开 `/canvas`，确认回到项目工作台，不进入旧画布库。
  3. 打开 `/image`、`/prompts`、`/assets`，确认统一使用左侧工作区导航。
  4. 在左侧栏点击项目工作台、生图工作台、提示词库、我的素材和 AI 配置。
- 通过标准：
  - 登录后的项目工作台是主入口。
  - 独立画布库旧入口不再干扰主流程。
  - 工具页入口、退出登录和 AI 配置入口可达。
- 失败影响：用户无法理解从哪里开始，或会被旧入口带离当前工作流。

### 2. 项目与分集生产入口

- 入口：`/projects`、`/projects/:id`。
- 操作：
  1. 新建项目，填写项目名称并选择预设。
  2. 打开项目详情页，点击“新建分集”并粘贴剧本文本。
  3. 导入后确认进入 `/projects/:id/episodes/:episodeId/workbench`。
  4. 从项目详情页返回项目工作台，再重新进入该项目和该分集。
- 通过标准：
  - 项目可创建、打开、重命名、归档或删除。
  - 分集导入后能直接进入本集生产流程。
  - 项目详情能表达当前制作进度和下一步入口。
- 失败影响：短剧生产闭环的起点失效。

### 3. 本集生产流程

- 入口：`/projects/:id/episodes/:episodeId/workbench`。
- 操作：
  1. 检查左侧阶段导航和当前阶段状态。
  2. 在导演分析、资产与生图、分镜生产包、画布承接各模块中分别打开详情或预览。
  3. 执行不会真实扣费的预览 / 确认 / 采用类操作。
  4. 在画布承接中创建或打开承接画布。
- 通过标准：
  - 阶段状态、阻塞原因和下一步操作清楚。
  - 结构化输出摘要可读，不直接暴露大量 JSON 字段。
  - 画布承接不会跳到旧抽屉流程。
- 失败影响：项目虽然可进入，但用户不知道下一步怎么推进。

### 4. 画布核心操作

- 入口：`/canvas/:id`。
- 操作：
  1. 从项目或本集流程打开已绑定项目 / 分集的画布。
  2. 验证画布拖拽、缩放、选择、框选、撤销、重做、删除、清空确认。
  3. 使用底部工具栏新增文本、图片、视频、音频和生成配置节点。
  4. 一次拖入多张图片 / 视频 / 音频，确认批量生成节点且不完全重叠。
  5. 点击“素材”和“本集生产流程”入口。
  6. 点击画布左上角返回入口。
- 通过标准：
  - 画布基础编辑稳定。
  - 已绑定分集时直接返回正式本集生产流程。
  - 未绑定上下文时有兜底绑定入口。
- 失败影响：核心创作载体不可用。

### 5. 素材管理与引用

- 入口：`/assets`、项目详情素材引用入口、画布素材入口。
- 操作：
  1. 新建文本、图片、视频、音频素材。
  2. 建立文件夹，移动素材，批量添加标签，导出 / 导入素材包。
  3. 将素材加入项目库，并从画布或分镜引用该素材。
  4. 替换素材形成新版本，检查旧版本下载、过期引用筛选和批量更新。
- 通过标准：
  - 素材本体、版本、项目库归属和引用关系可见。
  - 更新到最新版只修改引用方，不静默污染素材本体。
  - 本地文件缺失或旧版本引用能被发现。
- 失败影响：用户不敢替换、删除或复用素材。

### 6. AI 配置与模型边界

- 入口：左侧栏或顶部“AI 配置”弹窗、`/api/ai-config-template`。
- 操作：
  1. 下载配置模板，确认包含渠道、Base URL、API Key、模型、视频协议和思考模式字段。
  2. 修改模板后导入配置包，关闭并重新打开配置弹窗。
  3. 切换本地直连模式，开启 / 关闭思考模式。
  4. 使用缺失配置状态触发生图或画布生成，确认会提示先配置。
- 通过标准：
  - 配置包可导入、保存和恢复。
  - 生图接口不附加文本思考参数。
  - 本地直连和云端渠道边界符合 `docs/todo.md` 描述。
- 失败影响：AI 能力不可用或渠道边界混乱。

### 7. 生图工作台

- 入口：`/image`，以及本集生产流程“进入生图工作台”。
- 操作：
  1. 从本集资产提示词进入生图页，确认自动带入项目 / 分集 / 资产上下文和提示词。
  2. 添加本地参考图和剪贴板参考图。
  3. 检查左侧生成历史摘要加载速度，点击历史记录后再加载图片详情。
  4. 在配置完整的环境中生成图片，并保存到素材。
- 通过标准：
  - 生图页布局三栏清晰，右侧参数不被挤压。
  - 来源上下文能回写到素材 metadata。
  - 历史记录懒加载不影响回填。
- 失败影响：资产到生图、生成到素材的闭环断裂。

### 8. 视频 / 生产包承接

- 入口：本集生产流程“分镜生产包 / 画布承接”、`/video` 抽样。
- 操作：
  1. 在本集生产流程检查生产包状态、提示词、参考资产、风险提示和导入按钮。
  2. 导入生产包到承接画布，确认创建视频配置节点并聚焦。
  3. 打开 `/video`，确认它不是主生产入口，只作为低优先级审核 / 演示页存在。
  4. 在画布视频节点中检查时长、比例、音频开关和生成配置同步。
- 通过标准：
  - 正式视频链路从项目 / 分集上下文进入。
  - 独立 `/video` 不干扰主路径。
  - 视频时长不被强制锁定为 10 秒，音频默认关闭。
- 失败影响：用户可能把视频生产从项目上下文里做散，或生成参数错误。
