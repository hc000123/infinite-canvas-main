# 画布承接站与分镜检查时间线一期实施计划

## 目标

把画布从“前置生产入口”收束为“承接站”：上游本集生产流程完成审核后，画布负责承接分镜、提示词、素材和视频配置节点，并提供清晰的检查、微调和生成入口。

一期只做 UI 和操作层收口，不改变数据结构、不重写画布底层交互、不触发新的生成流程。

## 实施顺序

### 1. 右侧上下文检查器

新增或改造 `CanvasContextInspector`，作为右侧唯一主面板。

内容状态：

- 未选中节点：显示承接批次概览、缺口、待确认项、返回本集生产流程入口。
- 选中文本节点：显示完整文本、引用关系和改写 / 生图动作。
- 选中图片节点：显示图片、完整提示词、来源、保存素材、裁剪和多角度入口。
- 选中视频节点：显示视频、完整提示词、首尾帧、续写和任务状态。
- 选中配置节点：显示完整输入预览、模型、尺寸、数量和引用顺序。
- 需要 AI 助手时：把 `CanvasAssistantPanel` 作为检查器内的辅助 Tab 或折叠区。

### 2. 文字完整阅读规则

调整文本节点和配置节点展示：

- 节点内容默认换行，不用省略号隐藏关键文字。
- 节点高度设置合理上限，超长内容进入节点内阅读区。
- 右侧检查器显示完整输入、输出、提示词和引用详情。
- 系统提示、原始 JSON、规范读取和 debug 字段默认放到“记录 / 任务详情”。

### 3. 底部分镜检查时间线

在绑定项目和集数的画布中提供底部分镜时间线。

第一版只需要：

- 显示镜头卡片、标题、状态、时长和缩略图。
- 点击镜头后聚焦关联节点组。
- 右侧检查器显示该镜头完整脚本、Brief、提示词、参考素材和生成结果。
- 没有已审核分镜或 Seedance 提示词时，不默认展开时间线。

### 4. 承接入口文案统一

统一本集生产流程到画布的入口表达：

- “创建承接画布”：当前集没有画布时使用。
- “导入到画布”：已有画布，需要导入已审核结果时使用。
- 避免“新建一集画布并导入剧本”等前置生产文案。

导入确认页只做预览和取消选择，不自动生成图片或视频，不触发扣费。

### 5. 工具栏与悬浮工具降噪

调整 `CanvasToolbar` 和 `CanvasNodeHoverToolbar`：

- 底部工具栏只保留新增节点、上传、连接、配置、撤销重做、更多。
- 节点悬浮工具优先图标化，复杂操作移到右侧检查器。
- 删除、清空、提交审核等高风险操作保留确认。

## 验收标准

- 打开绑定集数的画布后，能看到承接状态和分镜检查入口。
- 未绑定集数的普通画布仍保持自由创作体验。
- 任意用户输入、AI 输出、提示词、脚本和分镜描述都能完整阅读。
- 主界面不直接展示系统推理过程、原始 JSON 或规范读取噪音。
- 从本集生产流程创建承接画布不会自动生成图片或视频。
- 选中节点后，用户能在右侧检查器完成主要查看和下一步操作。

## 不做事项

- 不改后端。
- 不改节点、素材、分镜、Agent 或任务日志的数据结构。
- 不实现新的模型调用或扣费逻辑。
- 不做移动端完整适配。
- 不把项目基准设定、剧本导入、导演分析和服化道拆解重新放回画布主界面。
